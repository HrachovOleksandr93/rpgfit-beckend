# Отчёт для партнёра — RPGFit, 18 апреля 2026

Привет! Вот простой отчёт о том, что мы продвинули сегодня. Без технических
терминов. Для человека, который понимает идею проекта и знаком с лором
(«Теория Мира v1.1»).

---

## 1. Что поменялось в идее по лору

Лор (шесть реалмов, Щель ВЕРА, Вектор-сканер) остаётся **каноном**. Но мы
отшлифовали то, как он переводится в игровую механику и пользовательский
опыт.

### Что убрали

- **5 рас (Человек / Орк / Дварф / Тёмный эльф / Светлый эльф) — убираем
  полностью.** Все игроки — люди. Причина: в нашей истории Щель пришла в
  **мир людей**, эльфов/орков у нас в каноне нет. Это также упрощает
  онбординг (меньше выборов для новичка) и балансировку боёв.
- **Фракционный раскол (Олимп vs Асгард vs …) — убираем.** Игрок **не
  выбирает** реалм как свою сторону. Никаких «+2% урона по чужим». Реалмы
  остаются только как **тема** — портал «из Асгарда» спавнит ледяных
  Йотунов, дропает артефакты Асгарда. Это лор, а не геймплейное деление.
  Причина: в нашем лоре всё человечество **вместе** против Щели, а не
  разделено на секты.

### Что расширили

- **Порталы.** Расширили концепт — теперь это не просто «точка на карте»,
  а два вида:
  - **Статические порталы** — в реальных культовых местах (Галдхьопиген в
    Норвегии, Теотиуакан в Мексике, Крымская Яйла, Днепровские пороги и
    т.д.). Пришёл физически — получи уникальный Tier 3 артефакт. Для тех,
    кто не может туда доехать — **виртуальная реплика** с упрощённой
    наградой.
  - **Динамические порталы** — временные Щели, спавнятся алгоритмом.
    Отложили на пост-бету.
- **Социальные события.** Раньше был только один глобальный ивент —
  «День Раскола» 31.10 (запуск беты). Теперь социалка — **приоритет
  номер один**. Минимум два глобальных ивента в бета-период (запуск +
  миди-бета босс через 3-4 недели). Добавили «живой фид» активности
  других игроков, счётчик активных операторов у каждого портала,
  share-карточки по итогам ивентов. **Чата, гильдий, друзей не делаем**
  — социалка через публичные ленты, не через модерируемые разговоры.

### Идеи в работе (ещё не решили)

- **Ментальные статы от данных mindfulness.** HealthKit и Health Connect
  отдают сон, медитацию, HRV, дыхание. Идея — привязать к «mental
  характеристикам» персонажа (WIS/INT/WIL/FOC), которые влияют на
  восстановление между боями, требования к топовому снаряжению и
  резистентность ментальных боссов.
  - **Вердикт агента-исследователя:** брать в бету только **один
    маленький кусочек** — сон → скорость восстановления. Полный 4-статный
    сплит откладываем на пост-бету (Q1 2027). Причина: Android Health
    Connect получил Mindfulness API только в конце 2025-го, и все
    известные фитнес-продукты (Whoop, Oura, Fitbit) схлопывают
    сон+HRV+дыхание в **один скор**, а не четыре. Ещё — есть
    репутационный риск с «геймификацией медитации» (критика
    «McMindfulness»).
- **Мобы таскают вещи из нашего мира.** Изначально предлагалась как
  lootмеханика (Apple Watch падает в инвентарь игроку). Уточнили с
  основателем: **это только визуально-дизайнерская фишка, не loot.**
  Часть мобов спавнится как «чемпионы» — визуально украшены нашими
  вещами (смартчас на руке Йотуна, гиря 16кг прикована к голове как
  броня), статы у них +25-30%. Игрок убивает их как обычных мобов,
  просто слегка потужее. Без отдельного инвентаря «реликты нашего
  мира».

### Что осталось без изменений

- Шесть реалмов как лор (Олимп, Асгард, Дхарма, Ду'ат, Нав, Шиба — канон
  надо финально сверить с docx).
- Vector как смартфон-как-сканер.
- XP за физическую активность (дневной кап 3000, формулы из
  BUSINESS_LOGIC).
- Битва как тренировка реального времени (HR → урон).
- Артефакты с привязкой к реалмам (Тризуб Посейдона — из Олимпа).

## 2. Что позаимствовали и откуда

Честно о том, что мы не придумываем с нуля:

### Из Ingress (Niantic)
- **Концепт статических порталов в реальных местах** — это прямо от
  Ingress.
- **Launch Anomaly events** (один день, все идут в свои города биться) —
  наш «День Раскола» построен по этой схеме. Паттерн проверен 10+ лет
  Niantic'ом.
- **Что НЕ взяли:** фракции Enlightened/Resistance. Нас это не устраивает
  по лору.

### Из Strava
- **KOM/QOM сегменты + Monthly Challenges + Kudos** — простая социалка
  через публичный фид без модерации. Наша «Active Players feed» и
  Weekly Recap построены по этому принципу.
- **Strava Live Segments 2025** (компактный банер «ты vs другие прямо
  сейчас») — взяли как паттерн для боевого экрана.

### Из Pokémon GO / Monster Hunter Now
- **POI маркеры с пульсацией на карте** — стандарт location-based
  геймдева. Наши Rupture dots на мапе по этому канону.
- **Короткие сессии боя (75 сек)** — MHNow ограничил и это оказался
  хороший UX для мобайла. Наши бои привязаны к длине реального сета
  — сопоставимо.

### Из Orna / Monster Hunter / D&D
- **Моб-класс иерархия** (I Слабые / II Средние / III Сильные / IV
  Элитные-боссы) — стандартный RPG-паттерн.
- **Инвентарь с rarity системой** (common → legendary, цвет борта
  определяет редкость) — WoW/Diablo канон.

### Из God of War / Hades / Smite
- **Подход к мифологиям как к «боги в конфликте с нашим миром»** — God
  of War 2018 показал, как Сони переосмыслил Асгард и Олимп в
  современной игре. Мы будем это использовать как референс для редизайна
  нашего бестиария мобов (агент-bestiary архитектор сейчас работает над
  этим — должен выдать 400-500 мобов с лор-текстами).

### Из Whoop / Oura
- **Ring-progress UI для метрик** и HRV sparkline — их стандарт, мы
  берём в дизайн 06 Neo Glass.
- **Единый "Recovery Score"** вместо кучи отдельных статов — их подход,
  именно поэтому мы решили не делать 4 ментальных стата сразу.

### Из Cyberpunk 2077 / Blade Runner / Ghost in the Shell
- **Signature motif «Corpo Redaction»** для дизайна 07 Night City
  (чёрные █████ бары, закрывающие classified-поля) — прямая отсылка
  к дизайнам Vilimovsky из CP2077. Но адаптировано под **украинский
  контекст** (тризуб, кириллица, ВЕРА-ІКС как корпо), а не generic
  Japanese cyberpunk.

### Из Fallout (только 05 Pip-Vault)
- Pip-Boy 3000 aesthetic — для дизайн-концепта #5.

### Из Spotify Wrapped
- **9:16 share-card формат, скроллируемые слайды** — канон. Наш
  Weekly Recap построен по этому паттерну.

### Что придумали сами
- **Signature motif «Rift Lens»** для Neo Glass — тонкая диагональная
  трещина на главной glass-панели. Ничего подобного у конкурентов нет.
  Прямо переводит «Щель ВЕРА» в UI.
- **Champion-моби с вещами из нашего мира** — Apple Watch на Йотуне
  как визуал. Уникально-мемабельно.
- **6 реалмов одновременно с общим врагом, без фракционного раскола**
  — отличает от Ingress (где разделение — core).

## 3. Что имплементировали в код сегодня

Сегодня **физического кода в приложении или бекенде не трогали**. Все
изменения в уровне **дизайна, документации и исследования**. Это
намеренно — перед тем как писать миграции БД и новые фичи, надо
зафиксировать решения, чтобы не переписывать потом.

### Что поменяли в коде/файлах

1. **Запустили Expo Web preview** (в клиентском приложении). Теперь
   React Native приложение можно открыть в браузере на
   `http://localhost:8081` и тестировать через Playwright (это нам
   нужно для автоматизированных проверок UI в будущем). Технически —
   добавили `react-dom`, `react-native-web`, `@expo/metro-runtime`,
   и включили `web` как платформу в `app.json`.

2. **Дизайн-директория обновилась:**
   - Из 10 концепт-заготовок осталось **7 проработанных** (удалили
     пустые). Каждый вариант имеет 4 экрана (было 3 у первых двух) —
     Battle, Map, Character, Inventory.
   - **06 Neo Glass** и **07 Night City Noir** — ваши фавориты —
     расширили до **9 экранов каждый** (добавили Onboarding 60 сек,
     Event «День Раскола», Weekly Recap 9:16, Active Players feed,
     Settings/Health permissions).
   - Исправили **2 WCAG accessibility FAIL**-проблемы (контрасты,
     которые не проходили стандарт AA для нормального текста).
   - Добавили **signature motifs**: «Rift Lens» (06) и «Corpo
     Redaction» (07) — фирменные визуальные элементы для узнаваемости.
   - Добавили в shared/base.css **reduced-motion** поддержку (для
     пользователей, чувствительных к анимации) и focus-visible
     outlines (доступность клавиатурой).
   - Убрали из кода Web3/NFT лексику («nft-card» → «relic-card»),
     чтобы не ассоциироваться с крипто.
   - В 07 добавили украинскую идентичность (кириллица «ВЕРА-ІКС»,
     трипільські орнаменти, синьо-жовтий акцент азура) вместо
     generic Japanese cyberpunk.

### Что НЕ меняли (намеренно)

- Backend (`rpgfit-beckend/`) — ни одной строки. Ждём, пока
  архитектурный агент прочитает bestiary и решения по ментальным
  статам/чемпионам, потом пишем миграции.
- React Native приложение (`rpgfit-app/src/`) — кроме web-настройки
  ничего. Много экранов из дизайна ещё ждут реализации.
- Формулы HP/XP в мобах, level curves — без изменений.
