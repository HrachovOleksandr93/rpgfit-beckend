# Beta Hype — Чорновик

> Цей файл — місце збору сирих ідей фаундера про те, як RPGFit має виходити в
> beta. Остаточна стратегія — у `BA/outputs/02-beta-scope.md` (агент 02 його
> генерує з цього + зі звіту 01).

## Ціль beta

Зібрати **перші 10k активних користувачів** і створити розмову про продукт у
fitness/RPG/geek спільнотах. Критерій успіху — 50%+ D7 retention + органічний
трафік з Reddit / TikTok / Twitter.

## Гіпотези, які хочеться перевірити в beta

1. Люди тренуються частіше, якщо тренування дає **RPG-прогрес з наративом**
   (не просто бали, а перемога над Йотуном з Асгарду).
2. Статичні портали у відомих місцях дають виклик для активних юзерів
   (Strava-audience).
3. Portal Creation Kit створює «моє місце» ефект → D30 retention вищий.
4. Realm-based моби формують фракційну лояльність («я за Олімп»).

## Чернетка хайп-механік

- **Launch event «День Розколу»** — приурочити реліз до 31 жовтня (лорний
  День Розколу) — масова рекламна кампанія, community event: у цей день
  з'являються спец-моби всіх реалмів.
- **Артефакти як collab** — умовно: перші 1000 юзерів на Галдхьопіген
  отримують оригінальний Мйольнір (можна чисто цифровий).
- **Реальні спонсорські challenge-и** — партнерство з спортзалами / паркрунами.
- **Реферальна програма** — ти приводиш друга, обидва отримуєте Portal
  Creation Kit.

## Таргет-аудиторія (гіпотеза)

- **Primary:** 20–35 років, чоловіки, геймери + fitness-amateurs.
- **Secondary:** 25–40, жінки, активні yoga/running.
- **Tertiary:** core RPG-геймери (Orna, Diablo, PoE) — прийдуть за лутом.

## Що треба з'ясувати у BA-агентів

- (01) Які community-days / launch events спрацювали у Pokémon GO, Orna?
- (01) Які партнерства (Strava × *) працюють для fitness-продуктів?
- (02) Який мінімальний контент (портали, моби, артефакти) потрібен, щоб
  перші 100 юзерів не «вийшли» через content-starvation.
- (05) Який сюжет-hook ми запускаємо у рекламу — «Боги повернулись»?
  «Ти — оператор Вектора»? «Твої тренування рятують світ»?

## Ідеї для post-beta (не в scope для beta)

- PvP-раїди між реалмами (гільдії).
- Season pass (90-денні сезони з унікальним лутом).
- AR-камера на статичних порталах.
- Мерчандайз (фізичні «Vector» device-реплікації для launch).
